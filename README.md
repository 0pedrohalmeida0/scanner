# Network Vulnerability Scanner (Python Multithreaded)

[PT-BR]
Este projeto é um motor de varredura de rede (Port Scanner) de alta performance, desenvolvido em Python. O foco principal foi a criação de uma ferramenta de auditoria de rede robusta e autocontida, utilizando conceitos fundamentais de engenharia de software e redes, sem depender de bibliotecas externas complexas.

**Destaques Técnicos:**
* **Socket Programming:** Comunicação direta de baixo nível via TCP.
* **Multithreading:** Gerenciamento paralelo de tarefas com `Queue` para otimização de performance.
* **Persistência de Dados:** Integração com SQLite para armazenamento estruturado de resultados e histórico.
* **Configuração Flexível:** Uso de arquivos `.env` para gestão de parâmetros, seguindo boas práticas de desenvolvimento.

---

[EN-US]
This project is a high-performance network scanner developed in Python. The core focus was to create a robust network auditing tool, utilizing fundamental software engineering and networking concepts without relying on heavy external frameworks.

**Key Technical Highlights:**
* **Socket Programming:** Direct, low-level TCP communication.
* **Multithreading:** Parallel task management using `Queue` for performance optimization.
* **Data Persistence:** SQLite integration for structured result and audit history storage.
* **Flexible Configuration:** Usage of `.env` files for parameter management, following industry best practices.
